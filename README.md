# py--study

## 좀이따가 재밌다 파일에 통합시킬 예정 지금은 귀차나

소스코드 블록은 1옆에 저거 3개 붙혀야 나온댕
```py
print("잠와죽겠다")
```

링크는 중괄호 넣어서
[블로그 주소](사이트 url)

순서 없는 목록은 별표 넣어서

* py 어쩌구저쩌구
  * 어쩌구 저쩌구
  * 뭐시기 거시기
    * 이러쿵 저러쿵
    
 인용구문은 >로
 > 잠와죽겄다
 
 테이블 작성도 가능
 
 가|나|다|라|
 ---|---|---|---|
 잠온다|정말|잠온다|하
